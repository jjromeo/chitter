
Chitter
=======

This is a project in which we were instructed to recreate the popular social network Twitter

Technologies used include:
-----------------------
- Ruby
- Sinatra
- Capybara
- Postgresql
- Haml
- Css

How to use: 
-------------
1. Git clone repo 
2. Bundle Install
3. ruby server.rb
4. visit localhost (default port 4567)
